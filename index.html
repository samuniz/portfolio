Super Syntax Tutor

Super Syntax Tutor is a Full stack MERN web application that allows users to create an account, choose a game with the goal of training the typing syntrax of specific web language.

This app is build using React, Express in conjunction with Node.js to create the server. Elements are stored in a Mongo database using Mongoose. Passport is used to create the user login function.

GitHub Deployed
Book Worm

Bookworm is a online app that keeps track of books you've read, are reading, or would like to in the future. You can keep track of your progress in a book, take notes on what you're reading as well as search for information on any book in the google books api to find something for future reading.

This app uses Express in conjunction with Node.js to create the server. Elements are stored in a MySql database, passport is used to create the user login function. Handlebars.js is the view engine and renders the JSON responses from the server. Bootstrap supplies most of the CSS. This application follows the MVC design pattern.

GitHub Deployed
Radical Reads Scrape

Radical Reads Scraper is a web scraper that displays the most recent aerticles from https://radicalreads.com/ . Users can save articles to the favorites list and leave comments on the favorites articles.

This app uses Express in conjunction with Node.js to create the server. Cheerio and Axios scrapes the site and the targeted elements are stored in a Mongo database using Mongoose. Handlebars.js is the view engine and renders the JSON responses from the server. Materialize supplies most of the CSS. This application follows the MVC design pattern.

GitHub Deployed
