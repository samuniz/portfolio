About Me

I'm a Full Stack web developer, passionate about creating useful and clean web applications.

My journey into tech began when I worked as a project manager at Yaskawa Motoman Robotics. I was very intrigued how code could give purpose to those machines and how powerful coding actually is. In 2017 I began to teach myself code because I wanted a more fulfiling career. My passion for code just keep growing and the more I learned, the more pasionate I became. In October 2019, I attendend the Full Stack Code Bootcamp at Penn University. I learned valuable, powerful, technical skills and the MERN stack as well as working in a team enviroment. I'm very passionate about tech, how it is changing the world and how it is challanging me. Teamwork, organization and communication are key components to my work style. I look forward to working in a team enviroment and building web applications that meet excede clients expectations.
